HRI 2026の論文127本を眺める

2026年7月1日 / HRI / Paper Analysis

ヒューマンロボットインタラクション分野の主要国際会議「HRI 2026（ACM/IEEE International Conference on Human-Robot Interaction）」が2026年3月16～19日，英国スコットランドのエディンバラで開催された。

HRIは，人とロボットの相互作用を扱う国際会議である。ロボットの認識・制御・学習といった技術だけでなく，人間の行動，信頼，ユーザー体験，デザイン，社会実装，倫理などを含めて議論する点に特徴がある。ロボティクス，人工知能，ヒューマンコンピュータインタラクション，ヒューマンファクター，心理学，社会・行動科学など，複数の分野の研究者が集まる学際的な会議である。

Google Scholar Metricsでは，HRIは「Human Computer Interaction」分野でh5-index 52を記録し，同分野の上位20件中13位に入っている。さらに「Robotics」分野でもh5-index 52で19位に入り，人とロボットの相互作用を扱う代表的な発表の場となっている。

今回のHRI 2026では，Full Paperに537件の投稿があり，128件が採択された。採択率は約24％だった。本記事では，Conference PublishingのHRI 2026 abstractsページから取得したFull Research Papers 127件のタイトルと概要をもとに，採択論文から見える研究トピックの分布を眺める。

単一の流行語より，実環境への広がりが目立つ

HRI 2026では，単一の支配的キーワードだけで全体像を説明するのは難しい。ケア，教育，公共空間，アクセシビリティ，遠隔操作，信頼，倫理，LLM，ロボット学習が互いに混ざりながら広がっている。

LLMや自然言語処理は確かに目立つ。ただし，HRI 2026では「LLMそのもの」が主役というより，会話設計，発話方略，チューターロボット，指示理解，意図推定，公共HRIの中に入り込む道具として現れている。技術単体というより，ロボットが人間の生活空間に入ったとき，どう振る舞うべきかを問う論文が多い。

主要なクラスタ

クラスタリング結果は，大きく次の8群として読める。

  • language agents / safety / technical systems: VLM/LLM，言語，ベンチマーク，安全，ハンドオーバーなどが混ざる技術寄りの群。
  • robot learning / demonstrations / feedback: デモンストレーション，制約，フィードバック，ポリシー学習が中心。
  • trust / social judgment / teaming: 信頼，協働，社会的評価，チーム行動に関する研究。
  • care / service / health deployments: ケア現場，サービス現場，医療・高齢者施設での導入に関する研究。
  • navigation / accessibility / user preferences: 移動支援，視覚障害者支援，ユーザ嗜好，環境内行動に関する研究。
  • robot form / design artifacts / public space: ロボットの形態，デザイン素材，公共空間での存在感に関する研究。
  • children / education / emotion learning: 子ども，教育，感情学習，チューター/ピアとしてのロボット。
  • methods / reporting / ethics / in-the-wild practice: 研究方法，報告標準，バイアス，実環境での設計実践。

横断テーマで見る

キーワードベースで粗く数えると，次のテーマが目立った。

49件collaboration / intent / task

43件ethics / safety / policy

40件trust / social perception

35件LLM / language / conversation

35件public / navigation / in-the-wild

25件children / education

24件assistive / care / health

16件teleoperation / operator

この数え方はキーワードベースなので厳密な分類ではない。それでも，HRI 2026が「ロボットをどう動かすか」だけでなく，「人間の判断，信頼，権力関係，ケア，教育，公共空間の中でロボットをどう成立させるか」に広がっていることは見えてくる。

この分析は，タイトルと概要だけを使った軽量な探索である。本文精読や引用関係の分析まではしていないので，結論は「研究動向の初期地図」として読むのがよい。

まとめ

HRI 2026は，LLMやロボット学習のような技術トピックを含みつつも，中心にあるのは実環境で人とロボットがどう関わるかという問いだった。ケア，教育，公共空間，アクセシビリティ，倫理，信頼が同じ地図上に現れるのが，HRIらしい。

採択論文を俯瞰すると，HRI 2026は特定の技術名だけで語るよりも，人とロボットが出会う場面の広がりとして読む方が自然に見える。技術，評価，デザイン，社会実装が同じ地図上に並んでいること自体が，HRIという会議の性格をよく表している。
