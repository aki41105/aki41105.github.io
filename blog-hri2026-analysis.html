HRI 2026の論文127本を眺める

2026年7月1日 / HRI / Paper Analysis

HRI 2026のFull Research Papersを，タイトルと概要からざっと分析した。日経クロステックのCVPR分析記事のように，採択論文から「いま研究が集まっている場所」を読むことが目的である。

対象は，Conference PublishingのHRI 2026 abstractsページから取得できたFull Research Papers 127件である。予稿集PDFの前書きでは537本投稿，128本採択と書かれている一方，公式TOCからメタデータと概要を取得できたFull Research Paperは127件だった。そのため，この記事では取得できた127件を基準にした。

単一の流行語より，実環境への広がりが目立つ

CVPRであれば「VLA」のような技術名が前面に出ることがある。HRI 2026では，それに相当する単一の支配的キーワードは見えにくい。むしろ，ケア，教育，公共空間，アクセシビリティ，遠隔操作，信頼，倫理，LLM，ロボット学習が互いに混ざりながら広がっている。

LLMや自然言語処理は確かに目立つ。ただし，HRI 2026では「LLMそのもの」が主役というより，会話設計，発話方略，チューターロボット，指示理解，意図推定，公共HRIの中に入り込む道具として現れている。技術単体というより，ロボットが人間の生活空間に入ったとき，どう振る舞うべきかを問う論文が多い。

主要なクラスタ

クラスタリング結果は，大きく次の8群として読める。

  • language agents / safety / technical systems: VLM/LLM，言語，ベンチマーク，安全，ハンドオーバーなどが混ざる技術寄りの群。
  • robot learning / demonstrations / feedback: デモンストレーション，制約，フィードバック，ポリシー学習が中心。
  • trust / social judgment / teaming: 信頼，協働，社会的評価，チーム行動に関する研究。
  • care / service / health deployments: ケア現場，サービス現場，医療・高齢者施設での導入に関する研究。
  • navigation / accessibility / user preferences: 移動支援，視覚障害者支援，ユーザ嗜好，環境内行動に関する研究。
  • robot form / design artifacts / public space: ロボットの形態，デザイン素材，公共空間での存在感に関する研究。
  • children / education / emotion learning: 子ども，教育，感情学習，チューター/ピアとしてのロボット。
  • methods / reporting / ethics / in-the-wild practice: 研究方法，報告標準，バイアス，実環境での設計実践。

横断テーマで見る

キーワードベースで粗く数えると，次のテーマが目立った。

49件collaboration / intent / task

43件ethics / safety / policy

40件trust / social perception

35件LLM / language / conversation

35件public / navigation / in-the-wild

25件children / education

24件assistive / care / health

16件teleoperation / operator

この数え方はキーワードベースなので厳密な分類ではない。それでも，HRI 2026が「ロボットをどう動かすか」だけでなく，「人間の判断，信頼，権力関係，ケア，教育，公共空間の中でロボットをどう成立させるか」に広がっていることは見えてくる。

自分の研究との接点

遠隔操作者の判断ログから，ロボットの高レベル対話戦略を学ぶ，という関心で見ると，直撃する大きなクラスタは見つからなかった。ただし，周辺にはかなり近い論文が散らばっている。

  • Immersive Social Teleoperation Interface with Semi-automatic Ingroup Navigation for Intuitive Communication: 遠隔操作，社会的ナビゲーション，直感的コミュニケーションに近い。
  • What You Reward Is What You Learn: Comparing Rewards for Online Speech Policy Optimization in Public HRI: 公共HRIでの発話ポリシー最適化。対話方略学習にかなり近い。
  • Input Matters: How Telepresence Control Devices Affect Performance, Sense of Control, and User Experience: テレプレゼンス操作の入力設計。遠隔操作者データの性質を考える上で重要。
  • Don't Say That! Proactive Support for Appropriate Power Use with Avatar Robot: アバターロボットと適切な権力使用の支援。遠隔存在の振る舞い設計に接続する。
  • InterPReT: Interactive Policy Restructuring and Training Enable Effective Imitation Learning from Laypersons: 非専門家からの模倣学習。操作者判断を教師データ化する議論に接続しやすい。
  • Reframing Conversational Design in HRI: Deliberate Design with AI Scaffolds: AIを使った会話設計支援。対話方略設計の周辺研究として重要。
  • Open-Ended Goal Inference through Actions and Language for Human-Robot Collaboration: 行動と言語から人間の目標を推定する研究。対話開始・継続判断の理論的隣接領域。

この見方からは，現在の研究の位置づけも少し見えてくる。HRI 2026には，公共空間での発話ポリシー，テレプレゼンス，対話設計支援，模倣学習，意図推定に関係する研究がある。しかし，遠隔操作者が現場で行っている「誰に話しかけるか」「いつ話すか」「いつ終えるか」といった高レベルな対話判断を，ロボットの対話戦略学習に使う問題設定は，まだ明確な主流トピックにはなっていないように見える。

この分析は，タイトルと概要だけを使った軽量な探索である。本文精読や引用関係の分析まではしていないので，結論は「研究動向の初期地図」として読むのがよい。

まとめ

HRI 2026は，LLMやロボット学習のような技術トピックを含みつつも，中心にあるのは実環境で人とロボットがどう関わるかという問いだった。ケア，教育，公共空間，アクセシビリティ，倫理，信頼が同じ地図上に現れるのが，HRIらしい。

個人的には，遠隔操作者の判断を対話戦略として学習する研究は，この地図の中でまだ空いている場所に見える。近い研究は複数ある。しかし，それらをつないで「現場の人間の判断を，ロボットの高レベルな対話方略へ変換する」と言い切れる余地は残っている。

Source: Conference Publishing HRI 2026 abstracts. The analysis used a local reproducible script with title and abstract text as input.
